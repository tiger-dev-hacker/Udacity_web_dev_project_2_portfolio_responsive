Latest Projects

Popcorn - A movie app

Popcorn is a MERN-stack based web application that uses React and Tailwind CSS to display dynamic UI experiences for movie cards. It fetches movies from the TMDB API to populate the MONGODB database with all the movies requested by the user. It also uses ExpressJS as a backend to serve api services such as authentication, meta-data retrieval and real-time analytics of trending movies for users.

Resumind-AI based resume scanner

Resumind is a React, Tailwind CSS, and Claude based ATS (Applicant Tracking System) - based resume scanning application that generates a match score for a resume on providing a job description. The service allows the user to upload their resume and paste the job description and then uses the built-in LLM model to provide the best possible match score for the project.

Cryptocurrency Calculator

This is a Python-native app that uses Streamlit, and flask frameworks to predict the prices of the bitcoin and stock price calculator in the near future. The prediction is done by the LSTM (Long Short Term Memory) model to use different datasets from Yahoo finance to generate efficient prediction values to guide user investments decisions in the right direction.
